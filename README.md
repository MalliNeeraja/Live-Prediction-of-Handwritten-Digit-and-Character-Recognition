# ✍️ Live Prediction of Handwritten Digit and Character Recognition using Deep Learning

## 🎓 Academic Project

This project was developed as part of my B.Tech in Artificial Intelligence and Machine Learning. The objective of this project is to recognize handwritten digits (0–9) and English alphabet characters (A–Z) in real time using Deep Learning techniques. A Convolutional Neural Network (CNN) was implemented using TensorFlow/Keras, and OpenCV was used for image preprocessing and real-time prediction.

## 🚀 Features

* Real-time handwritten digit recognition
* Handwritten character recognition
* CNN-based deep learning model
* OpenCV image preprocessing
* Tkinter-based graphical user interface
* High prediction accuracy

## 🛠️ Technologies Used

* Python
* TensorFlow
* Keras
* OpenCV
* NumPy
* Pandas
* Tkinter
* Matplotlib

## 📂 Dataset

* MNIST Dataset
* EMNIST Dataset

## ▶️ Installation

```bash

cd Live-Prediction-of-Handwritten-Digit-and-Character-Recognition
pip install -r requirements.txt

```
